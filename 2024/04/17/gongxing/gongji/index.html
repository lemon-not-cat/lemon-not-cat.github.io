# 二字词语积累

  • 【傲然】
    高傲、坚强不屈的样子。
    例:~ 挺立
  • 【傲物】
    骄傲自大，瞧不起人。
    例：恃才～，
  • 【傲人】
    (成绩等) 值得骄傲、自豪。
    例：业绩～
  • 【奥秘】
    深奥的尚未被认识的秘密。
    例：探索宇宙的～。
  • 【奥妙】
    (道理、内容) 深奥微妙。
    例:~ 无穷。
  • 【奥义】
    深奥的义理。
    例：深得其中～。
  • 【暴发】
    ①突然发财或得势。
    例:~ 户。
    ②突然发作。
    例：山洪～。
  • 【爆发】
    ①火山内部的岩浆突然冲破地壳，向四外迸出。
    例：火山～。
    ②突然发作；:(事变) 突然发生。
    例:~ 革命。
  • 【悖论】
    逻辑学指可以同时推导出或证明两个互矛盾的命题或理论体系。
  • 【悖谬】
    荒谬；不合道理，
  • 【背书】
    背诵念过的书。
    例：每天早晨要～，背不出书要挨罚。
  • 【背书】
    持有票据的人转让票据时，在票据背面批注并签名盖章。经过背书的票据，付款人不能付款时，背书人负付款责任。
  • 【本原】
    哲学上指 - 切事物的最初根源或构成世界的最根本实体。
  • 【本源】
    事物产生的根源。
    例：想象力是创造力的～之一
  • 【本体】
    ①德国哲学家康德唯心主义哲学中的重要概念，指与现象对立的不可认识的 “自在之物”
    ②机器、工程等的主要部分。
  • 【比肩】
    ①并肩。
    例:~ 作战。
    ②指相当；比美。
    例：他虽然是票友，水平却可与专业演员～。
  • 【比美】
    美好的程度不相上下，足以相比。
  • 【彼岸】
    ①(江、河、湖、海的) 那一边；对岸。
    例：大洋～。
    ②佛教指超脱生死的境界 (涅槃)
    ③借指所向往的境界。
    例：走向幸福的～。
  • 【彼此】
    ①那个和这个；双方。
    例：不分～。
    ②客套话，表示大家一样 (常叠用)。
    例:“您辛苦啦！” “~~！”
  • 【弊病】
    ①弊端。
    例：管理混乱，恐有～。
    ②缺点或毛病。
    例：制度不健全的～越来越突出了，
  • 【弊端】
    由于工作上有漏洞而发生的损害公益的事情。
    例：消除～。
  • 【弊書】
    弊病；害处。

# 四字成语积累

  • 【不可或缺】
    非常重要，不能有一点点的缺失，不能少一点。
    例句：具有高度专业水平的内容对任何媒体都是不可或缺的，这是媒体的安身立命之本。
  • 【举足轻重】
    脚稍微移动一下，就会影响两边的轻重；指处于重要地位，一举一动都足以影响全局。
    例句：李勇在公司里的地位举足轻重，任何决策都需要听取他的意见。
    成语区别：不可或缺强调存在重要而唯一性，举足轻重指只是相对比较重要，没有唯一性而言。
  • 【志得意满】
    意思是志向实现，心满意足。
    例句：党员领导干部要把名利看淡一点，不因提拔志得意满，不因离退心理失衡。
  • 【踌躇满志】
    形容对自己取得的成就非常得意。
    例句：协会主席大兵踌躇满志，要将 “笑工厂” 打造成湖南的 “德云社”
    成语区别：志得意满侧重于强调志向已然实现，心满意足：踌躇满志侧重于强调将要取得的成就得意
  • 【自惭形秽】
    意思是因为自己不如别人而感到惭愧。
    例句：每当看到那些为社会做出巨大贡献的人，我总是会为自己的渺小和无力而自惭形秽。
  • 【自暴自弃】
    自己瞧不起自己，甘于落后或堕落。
    例句：偶尔的自暴自弃不妨视为一种享受，至少有益于精神上的健康。
    成语区别：自惭形秽侧重于因为实际原因，自己能力比不上他人而感到惭愧。自暴自弃侧重于不知自爱，既可指心理状态，也指行动表现，语意比较重。
  • 【未雨绸缪】
    在天还没下雨的时候，就修补好房屋的门窗。后比喻事先做好准备。
    例句：未雨绸缪地调整生育政策，对于未来中国人口均衡发展可以产生积极的效果，
  • 【防患未然】
    意思是指在祸患发生之前就加以预防例句：安全检查工作非常重要，要防患未然，一旦出了事故，后悔也就来不及了
    成语区别：未雨绸缪是比喻性的，而防患未然是直陈性的。后者有 “防患” 两字，语义较前者清楚明确。防患未然偏重于防，未雨绸缪偏重于备。
  • 【干帆竞发】
    许多船只争着出发前行，形容声势浩大，也寓意竞争激烈。
    例句：千帆竞发，几经风雨几经浪；万马奔腾路凯歌一路春。
  • 【百家争鸣】
    比喻各种不同形式和风格的艺术自由发展。也形容艺术界的繁荣景象
    例句：春秋时期出现了百家争鸣的局面。
    成语区别：千帆竞发侧重形容竞争激烈，适用范围；百家争鸣侧重形容文艺界的繁荣景象。
  • 【空中楼阁】
    悬在半空中的阁楼。比喻虚幻的事物或脱离实际的空想。
    例句：脱离国情和实际的立法，即使搞出来，也只是空中楼阁，解决不了实际问题。
  • 【海市蜃楼】
    屋：原指海边或沙漠中，由于光线的反向和折射，空中或地面出现虚幻的楼台城郭。现多比喻虚无缥渺的事物。
    例句：他对未来的规划总是那么美好，，但如果不付出努力，那些梦想就只会是海市蜃楼。
    成语区别：海市蜃楼侧重比喻虚无缥渺的具体事物空中楼阁侧重比喻脱离实际的理论、计划及空想。
  • 【化险为夷】
    意思是化险阻为平易。
    例句：我们遭遇过很多外部风险冲击，最终都能化险为夷，靠的就是把发展立足点放在国内。
  • 【绝处逢生】
    形容在最危险的时候得到生路。
    例句：这一军事谋略创造了人民军队屡次绝处逢生力挽狂澜的奇迹。
    成语区别：化险为夷强调通过积极努力化解危险的过程和结果，而绝处逢生则强调在绝境中意外获得生机或希望的情境。
  • 【变幻莫测】
    指事物变化很多不能预料，变化多端难以揣测。
    例句：股市行情变幻莫测，投资者需有强大的心理承受能力和敏锐的市场洞察力。
  • 【瞬息万变】
    指在极短的时间内变化快而多。
    例句：科学研究工作者应该对瞬息万变的国内外形势作出精准预测和判断。
    成语区别：变幻莫测侧重于强调变化多，难以捉摸瞬息万变强调短时间内变化速度快。
