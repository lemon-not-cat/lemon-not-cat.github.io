# 概念间的逻辑关系

  1. 全同关系
  • 概念相同，说法不同 如：番茄 - 西红柿、土豆 - 马铃薯
  • 书面语、口语（雅俗） 如：那有两只老鼠 - 那有两只耗子
  • 古今不同 如：南京 - 金陵、医生 - 郎中
  • 中外不同 如：email - 电子邮件、引擎 - 发动机
  • 一语双关 如：炒鱿鱼 - 解雇、和稀泥 - 调解
  • 自他 如：令郎（你的儿子）- 犬子（自己的儿子）
  1. 全异关系
  • 绝对反义 -- 矛盾关系 如：男女、生死
  • 相对反义 -- 并列（反对）关系：同一大类里的两个小类 如：钢笔 - 铅笔（并列）
  1. 交叉关系
  • 有的 A 是 B，有的 A 不是 B，有的 B 是 A，有的 B 不是 A
    例：青年 - 教师 | 学生 - 党员
    注：当命名方式不同 80% 可能会出现交叉
    如：木桶、圆桶、保温桶
  • 出现命名方式：80% 是交叉，20% 是并列
  1. 包含关系

  • A. 种属（造句：…… 是…… 的一种）
    例：虎 - 豹 - 狮 - 猫科动物

  • B. 组成（造句：…… 是…… 的一部分）
    例：电脑 - 主板 - cpu - 显卡 - 硬盘 - 显示器

# 语义关系

  1. 近义

  2. 反义：同一话题

二级辨析：

  • （1）感情色彩：褒贬义（积极、消极）
  • （2）程度轻重
  • （3）拆构（冷嘲热讽、寒耕暑耘、东张西望）

  1. 比喻义 如：眉目、耳目

  2. 象征义

# 语法关系

主语 +（状语）谓语 +（定语）宾语

  • 主谓关系（主语 + 谓语，名词 + 动词）
  • 动宾关系（谓语 + 宾语，动词 + 名词）
  • 主谓宾 n + v + n
  • 主宾
  • 偏正关系 （形容词 + 名词， 副词 + 动词，名词 + 名词）

# 平面图形

一般有四类：位置类、样式叠加类、属性类、数量类

# 平移

一般是白色背景黑色块，或者白色背景黑色⚪

  1. 方向
  2. 距离
  3. 方式：循环移、往返 “回” 字形循环

# 旋转

各元素组成完全相同，局部位置变化明显

  1. 方向：顺逆时针
  2. 角度：30、25、60、90、180 度
  3. 方式：整体转、部分转

# 翻转

各个元素组成完全相同，局部位置变化明显

  1. 上下翻转（横轴对称）
  2. 左右翻转（竖轴对称）
  3. 整体反转、部分翻转
